GSMA Validation Quick Checklist

  • ✅ contentMessage wrapper is present
  • ✅ Valid JSON (no trailing commas, proper quotes)
  • ✅ Text ≤ 3072 characters, UTF-8
  • ✅ cardOrientation is VERTICAL or HORIZONTAL
  • ✅ Title ≤ 200 chars, Description ≤ 2000 chars
  • ✅ Media height: SHORT, MEDIUM, or TALL
  • ✅ fileUrl uses HTTPS (HTTP rejected)
  • ✅ Image max 3.5MB JPEG, 1MB PNG
  • ✅ Video max 100MB, MP4 H.264
  • ✅ Max 4 suggestions per message/card
  • ✅ Suggestion text ≤ 25 characters
  • ✅ postbackData ≤ 200 characters
  • ✅ No duplicate postbackData within message
  • ✅ messageId is unique (UUID recommended)
  • ✅ Brand registered with carrier
  • ✅ Fallback SMS message prepared
